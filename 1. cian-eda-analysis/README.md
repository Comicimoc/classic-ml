# Анализ данных ЦИАН по аренде квартир в Нижнем Новгороде

## Описание
Репозиторий "**cian-eda-analysis**" содержит исследовательский анализ данных (EDA) по объявлениям аренды квартир в Нижнем Новгороде, полученным с помощью библиотеки `cianparser`. Это мой самый первый проект в области машинного обучения.

## Цели проекта

- Получить чистый датасет по аренде квартир в Нижнем Новгороде.
- Разобраться в качестве данных, структуре признаков и источниках ошибок парсинга.
- Выявить базовые зависимости.

## Изменения в библиотеке `cianparser`
Базовая версия библиотеки `cianparser` не работала, поэтому в нее внесены различные изменения. Например:
- Детект антибот-страниц (Yandex SmartCaptcha и др.)
- Проверка на пустые страницы с объявлениями
- Добавление случайных задержек для обхода антибота
И др.

## Стэк технологий
- Python
- Scikit-learn, NumPy, Pandas, Matplotlib (Seaborn)
- PyCharm, Jupyter Notebook

## Результаты
1. Датасет после парсинга оказался “грязным”: присутствуют пропуски, технические значения и артефакты парсера, которые пришлось обрабатывать
2. В ходе EDA выявлены базовые закономерности:
- цена зависит от количества комнат
- есть различия по районам
- распределение цен скошено и содержит выбросы
3. Попытка построить простую линейную модель показала слабое качество
4. Сделан вывод, что текущих данных недостаточно для качественного обучения модели. По итогу этот проект используется как демонстрация EDA.
