# Анализ данных ЦИАН по аренде квартир в Нижнем Новгороде

Этот репозиторий содержит исследовательский анализ данных (EDA) по объявлениям аренды квартир в Нижнем Новгороде, полученным с помощью библиотеки `cianparser`.

---

## 1. Цели проекта

- Получить чистый датасет по аренде квартир в Нижнем Новгороде.
- Разобраться в качестве данных, структуре признаков и источниках ошибок парсинга.
- Выявить базовые зависимости.

---

## 2. Изменения в библиотеке `cianparser`
Базовая версия библиотеки `cianparser` не работала, поэтому в нее внесены различные изменения. Например:
- Детект антибот-страниц (Yandex SmartCaptcha и др.)
- Проверка на пустые страницы с объявлениями
- Добавление случайных задержек для обхода антибота
И др.
